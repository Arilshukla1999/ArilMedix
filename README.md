🧠 ArilMedix – AI Symptom Checker & Doctor Finder

An intelligent web-based platform that helps users analyze their health symptoms using AI and find nearby doctors for further consultation.

---

🚀 Features

- 🤖 AI-powered symptom analysis
- ⚠️ Severity detection (Mild / Moderate / Serious)
- 💡 Personalized health advice
- 🚨 Emergency warning system
- 📍 Find nearby doctors using Google Maps
- 🎨 Clean and responsive UI

---

🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- AI API: Groq
- Deployment: Vercel

This application is not a substitute for professional medical advice.
Always consult a certified doctor for proper diagnosis and treatment.
